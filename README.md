# Rinconcito Frutal y Snacks POS

Progressive Web App (PWA) integral para la operación diaria de juguería, cafetería y comida con POS, Cocina, Entregas, Caja, Créditos y Admin.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Configure `.env.local` based on `.env.example`.
3. Run the app:
   `npm run dev`
